tech stack used are-NODEJS,REACTJS MONGODB,EXPRESSJS,GORAK LLM API,SOCKET.IO
A real-time chat application built with React, Node.js, Socket.io, and MongoDB, 
featuring two AI-powered utilities: chat summarization and natural language search over message history (powered by the LLM model ofGroq API).

Features
Join the chat instantly with just a nickname (no authentication required)
Real-time message broadcasting to all connected users
Chat history loaded automatically when a new user joins
AI-generated summary of the last 20–30 messages
Natural language search over chat history (e.g. "Who mentioned coffee?")
Loading states for all AI actions
